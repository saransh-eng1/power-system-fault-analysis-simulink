# power-system-fault-analysis-simulink
MATLAB/Simulink project for analyzing transmission line faults and implementing overcurrent relay protection in a three-phase power system.


Power System Fault Analysis and Relay Protection using MATLAB/Simulink

This project models a three-phase electrical power transmission system using MATLAB Simulink (Simscape Electrical). The system includes a generator, transformer, transmission line, circuit breaker, and load. Different fault conditions are simulated to study their impact on system current and protection response.
The project implements a simple overcurrent relay protection scheme. When the fault occurs, the relay detects excessive current and sends a trip signal to the circuit breaker to isolate the faulty section of the network.

Key Features
1. Modeling of a three-phase transmission system
2. Simulation of short-circuit faults
3. Implementation of overcurrent relay logic
4. Automatic circuit breaker tripping
5. Visualization of fault current waveforms using Simulink Scope

Tools Used
1. MATLAB
2. Simulink
3. Simscape Electrical

Learning Outcomes
1. Understanding of power system fault behavior
2. Basic implementation of protection systems
3. Analysis of current transients during faults
